🤖 Aircall Junk Contacts — Deep Dive

9,289 auto-created contacts from Aircall integration. Still growing in real-time. This was the #1 finding in the Notion audit too (originally counted 9,133 — it's grown since).
🔴

This database is growing right now — every unknown inbound call creates a new junk contact

The Aircall integration is configured to auto-create a HubSpot contact whenever someone calls in who isn't already in the system. The result: firstname = +13035341231, lastname = Aircall new contact, no email, no company. The fix is in Aircall settings — not HubSpot.

What These Contacts Look Like
Every Aircall junk contact has the same pattern:
firstname: +13035341231
lastname: Aircall new contact
email: (empty)
company: (empty)
phone: +13035341231
num_associated_deals: 0
hubspot_owner_id: (empty)
3-Step Fix
Permanent solution to stop Aircall from polluting your CRM
1

Delete all 9,289 existing junk contacts in HubSpot

Filter by Last Name = "Aircall new contact". Bulk-select all → Delete. These have zero recoverable value.

Do this week — Notion audit Priority #1
2

In Aircall: Disable auto-contact creation for unknown callers

Go to Aircall Dashboard → Integrations → HubSpot → Turn off "Create contact on new call." Unknown callers will still be logged as calls but won't create contacts.

Stops new junk immediately
3

Set up a smart workflow: Enrich within 24hr or auto-delete

If you want to keep phone-only contacts: HubSpot Workflow → trigger on contact created with lastname "Aircall new contact" → wait 24hr → if still no email → delete. If email found via enrichment → keep and update name.
